# Task Flow API

Projeto desenvolvido para prática de conceitos DevOps e backend utilizando containers, automação de pipeline e monitoramento de serviços.

---

## Objetivo

Simular um ambiente moderno de desenvolvimento contendo:

- API em FastAPI
- Banco PostgreSQL
- Containers Docker
- Pipeline CI/CD com Jenkins
- Monitoramento com Prometheus e Grafana
- Estrutura organizada para Terraform

---

## Tecnologias utilizadas

- Python
- FastAPI
- PostgreSQL
- Docker
- Docker Compose
- Jenkins
- Terraform
- Prometheus
- Grafana
- Git e GitHub
